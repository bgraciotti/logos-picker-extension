## [1.4.5] - 2026-02-10

### Fixed
- **Thumbnail display**: Fixed issue where logo thumbnails were not loading on Raycast Windows v0.44+. The fix converts Windows file paths to file:/// URLs with forward slashes, which is now required by Raycast.
